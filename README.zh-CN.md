# StatusWords

[English](README.md)

在 Obsidian 状态栏中显示可自定义的单词和字符统计。

StatusWords 会实时显示当前笔记、当前行、当前段落、选中文本、光标位置和文件信息相关的统计数据。

## 功能

- 显示当前行、当前段落、整篇笔记和选中文本的字符数。
- 显示选中文本行数、光标位置、文件名和文件路径。
- 支持通过模板自定义状态栏显示内容。
- 支持多种计数模式。
- 支持中文模板占位符，同时兼容旧版英文占位符。

## 安装

### 从社区插件目录安装

1. 打开 Obsidian 设置。
2. 进入 **第三方插件**。
3. 搜索 **StatusWords**。
4. 安装并启用插件。

### 手动安装

1. 从最新发布版本下载 `main.js`、`manifest.json` 和 `styles.css`。
2. 在你的仓库中创建 `.obsidian/plugins/status-words/` 文件夹。
3. 将下载的文件放入该文件夹。
4. 重启 Obsidian。
5. 在 **设置 > 第三方插件** 中启用 StatusWords。

## 使用方法

启用插件后，状态栏会显示当前笔记的统计信息。

打开插件设置，可以自定义显示模板、计数模式和数字千分位分隔。

### 模板示例

```text
当前行：{{当前行字数}} 字　全文：{{全文字数}} 字
全文 {{全文字数}} 字｜共 {{总行数}} 行
{{文件名}}｜第 {{光标行号}} 行，第 {{光标列号}} 列
```

### 可用模板值

- 当前行字数
- 当前段落字数
- 全文字数
- 选中文本字数
- 选中文本行数
- 文件名
- 完整文件名
- 文件路径
- 光标行号
- 光标列号
- 总行数

### 计数模式

- Obsidian 风格：空格、标点、换行都会计入字符数。
- 不计空白：不统计空格、制表符和换行。
- 文字与数字：只统计文字和数字，不统计空格、标点和符号。
- 可见字符：按肉眼看到的字符统计，emoji 等组合字符通常算 1 个。

## 隐私

StatusWords 不联网，不收集遥测数据，不向外部服务发送任何内容。插件只读取当前 Obsidian 编辑器中的文本，用于在本地计算并显示统计信息。
